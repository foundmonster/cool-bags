# Cool Bags

A searchable database of backpacks, duffels, slings, and accessories from top bag brands.

## Features
- Browse 101+ bags across 12 brands
- Filter by brand, material, and type
- Sort by price, size, or name
- Pin favorites to compare

## Brands
EVERGOODS, Wexley, Aer, Pakt, Black Mile, DSPTCH, Able Carry, Tortuga, Hyperlite Mountain Gear

---

Built with vanilla HTML, CSS, and JavaScript.
