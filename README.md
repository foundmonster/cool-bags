# Cool Bags

A searchable database of backpacks, duffels, slings, and accessories from top bag brands.

## Features
- Browse many bags across many brands
- Filter by brand, material, and type
- Sort by price, size, or name
- Pin favorites to compare
- Submit feedback and brand requests

## Technology Stack

- **Frontend:** Vanilla HTML, CSS, JavaScript
- **Hosting:** Netlify
- **Data:** JSON-based bag database
- **Functions:** Netlify Serverless Functions
- **APIs:** GitHub API (feedback), Buttondown API (newsletters)

## Local Development

```bash
# Install dependencies
npm install

# Run local Netlify dev server
npm run dev
```

Visit `http://localhost:8888` to preview the site with Netlify Functions support.

## Deployment

Deployed on Netlify with automatic deployments from the `main` branch.
