## Download Workflow

For each brand:

1. **Research brand URL** (if not already known)
2. **Navigate to bags/backpacks collection page**
3. **Use WebFetch to extract product data:**
   - Product names
   - Prices
   - Volumes/capacities
   - Materials
   - Image URLs
4. **Download images to brand folder:**
   ```bash
   curl -o brandname-productname.jpg "https://image-url"
   ```
5. **Document any issues** in download-log.md

## Expected Challenges

**Known Issues from Previous Extractions:**
- Dynamic JavaScript sites (DB, London Bridge Trading) - WebFetch won't work
- Missing volume data - estimate based on similar bags
- Badge/award overlays - check for alternate images
- Dark backgrounds - prefer light-colored bag variants
- White bags on white backgrounds - use dark variants
- CDN access restrictions - may need alternate sources
